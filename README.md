# ☠ Ses-Sese
<quote>☠ This is the public voting website designed to be used for the events, such as Kelle-Kelleye in order to keep the justice.</quote>

### Deployed:
https://sessese.woosal.com/

### Used Technologies:
```
React JS
Node.js
Firebase
```

### Features:
```
1) No Authentication is required
2) Users can only choose one option and not change their answers (at least for now)
3) IP Blocked - no more than one vote is allowed using the same IP/Device
4) No special cookies are saved
5) Completely free of charge
```

### Additional Features:
- [] Mobile Friendly
- [] Full Firebase Integration
- [] IP Block/Limit
- [] Live Data / Who is winning visualization page || Feature
- [] Deployment
